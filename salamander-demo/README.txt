Salamander Grand Piano — demo excerpts
=======================================

These four .wav files are velocity layer v8 notes from Salamander Grand Piano
(Yamaha C5), included here only as a small demo for Ambient Samples.

Original library: Salamander Grand Piano by Alexander Holm.
License: CC BY (see the full library README and SFZ).

Renamed for URLs in this repo:
  D#4v8.wav  → Dsharp4v8.wav
  F#4v8.wav  → Fsharp4v8.wav

Notes: C4, D#4, F#4, A4 (same four suggested for ambient layering).
