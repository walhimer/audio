Sync (Sequencer)

Flow lanes — up to four voices run in parallel. Each step, every lane may fire based on its density and pitch rule (fixed scale row or random). Set how each voice behaves, adjust the mix, press Play, and let the loop run. Uses Tone.js; local Salamander WAVs go in salamander-local/ (v10 layer), or use web MP3s from tonejs.github.io.

Step —

Voices

Density = chance this lane plays each step. Pitch = which row of the scale (low → high), or random within the scale.

Library: tone.js next to this file. Local piano: salamander-local/*.wav (Salamander v10 layer).
