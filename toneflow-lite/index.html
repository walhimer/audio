Flow Lite

Sequencer using Tone.js. Your Salamander Grand Piano V3 pack (48khz24bit WAVs, CC-by) uses many velocity layers per note (C4v1.wav…v16). This app copies one layer (v10) per note into salamander-local/ and maps the same minor-third grid as the Tone.js MP3 demo. “Piano (web)” uses small MP3s from tonejs.github.io instead (no large local files).

Library: tone.js next to this file. Local piano: salamander-local/*.wav (Salamander v10 layer).
